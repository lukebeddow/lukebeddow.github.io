# Website for Luke Beddow

I am a robotics and machine learning researcher with a PhD in robotic manipulation.

### Setup

Dependencies:
* Node.js: https://nodejs.org/en/download, check `node -v` and `npm -v` both work.

Install:
* Clone and run `npm install`

Compile and edit:
* Begin the website locally with `npm start`
* Build changes with `npm run build`

Deploy online:
* Use `npm run deploy`
